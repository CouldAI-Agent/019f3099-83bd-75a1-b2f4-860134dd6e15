# Professional CV App

A professional, one-page, ATS-friendly Curriculum Vitae built with Flutter.

## Overview
This application renders a clean, digital CV styled as an A4/Letter document. It uses a single-column layout with clear headings which is optimized for Applicant Tracking Systems (ATS) readability. 

The content is currently pre-filled with professional placeholder data tailored for a **Chemist** applying for roles in the **Pharmaceutical industry**, specifically highlighting data management skills like **Microsoft Excel** and **MS Office**. 

## Features
- **ATS-Friendly Layout:** Single column design with standard bullet points and clear dividers.
- **Responsive Design:** Functions beautifully on web/desktop (displaying as a floating paper document) and mobile (filling the screen naturally).
- **Comprehensive Sections:** Includes Contact Info, Career Objective, Skills, Work Experience, Education, Internships, Certifications, Languages, Achievements, and Hobbies.

## Getting Started

1. Ensure you have [Flutter](https://docs.flutter.dev/get-started/install) installed.
2. Clone this repository.
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter run` to start the app on your preferred device or web browser.
